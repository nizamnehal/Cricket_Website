# 🏏 Cricket Website

A modern and responsive **Cricket Website** built using **HTML, CSS, Tailwind CSS, and JavaScript**.
This project showcases a clean sports-themed UI with responsive layouts, team information, and gallery sections designed using modern frontend practices.

## ✨ Features

* ✅ Fully Responsive Design (Mobile + Tablet + Desktop)
* ✅ Modern Cricket-Themed UI

* ✅ Team Section
* ✅ Gallery Section
